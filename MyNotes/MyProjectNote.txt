Now go to the Terminal, run the [ tsc ] command.
It will compile .ts files from the src folder and will create respective converted .js files in the dist folder.

If everything works fine, then run the command [tsc -w] in the Terminal.
It will watch all .ts files for any changes and will immediately create respective converted .js files in the dist folder.
